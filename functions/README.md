# Чистые функции

## Легенда
Во время игры необходимо будет отображать полоску жизни над игровым персонажем. Для сигнализации пользователю вы решили ввести цветовую индикацию:

Здоровье более 50 - зелёный;

Здоровье от 50 и до 15 - жёлтый;

Менее 15 - красный.

## Описание
Реализовать функцию, которая на вход принимает объект вида:

{name: 'Маг', health: 90}

И возвращает ответ в виде одной из строк: 

healthy, wounded, critical

## Задача
Сгенерировать проект на базе npm. Подключить туда jest и написать авто-тесты, которые обеспечивают 100% покрытие функции по строкам. Убедится, что обеспечил 100% покрытие тестами.

## Важно: 

ESLint не должен выдавать ошибок. 
Jest должен обеспечивать 100% покрытие по строкам для тестируемых функций.

Ко всем задачам должен быть подключен GitHub Actions и выставлен бейджик статуса.
